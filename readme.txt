RSS-to-post will add a items from an RSS Feed to the bottom of your post content.

== Description ==

This plugin will add items from an rss feed to your blog.
The items will show after the content on your single post.
The will pull new content every time the page loads thus
helping with Google Rank since you will have new content on your
single post pages.

 Enjoy! 
   Gordon French

== Installation ==

 1. Upload the full directory into your wp-content/plugins directory
 2. Activate it in the Plugin options
 3. Set a default feed and enable the plugin under rss-to-post settings tab

== Frequently Asked Questions ==

= Can I change the colors =

The colors get there setting from your active theme, however you can change them yourself 
by simply add these classes to your style sheet and change as you see fit.
.rssEntree {}
.rssTitle {}
